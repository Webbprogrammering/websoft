Explain the role of HTML, CSS and JavaScript in web development.

Those three languages combined are used to present the information to the user through HTML, style the user interface through CSS and finally make the website dynamic with the use of JavaScript.

Give a brief explanation of how the browser, the HTTP protocol and the web server interacts.

The browser's work is to load the necessary files, render them for the user and allow for user interaction. Through the HTTP protocol it is able to ask for those files from remote servers, and download the files that as mentioned before are rendered for the user. These files include the HTML, the CSS and even the JavaScript files that the website consists of.
